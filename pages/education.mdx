# `GET/user/education`

This endpoint returns all the education entries the user has created.

## Basic Information
- type: `REST`
- method: `GET`
- url: https://www.api.remoet.dev/user/education
- returns: `JSON`

## cURL
```bash
curl --location 'https://www.api.remoet.dev/user/education' --header 'Authorization: Bearer <Your key>'
```

## HTTP
```http
GET /user/education HTTP/1.1
Host: www.api.remoet.dev
Authorization: Bearer <Your key>
```

## JavaScript Fetch
```javascript
const myHeaders = new Headers();
myHeaders.append("Authorization", `Bearer ${key}`);

const requestOptions = {
  method: "GET",
  headers: myHeaders,
};

fetch("https://www.api.remoet.dev/user/education", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.error(error));
```

## DTO
```typescript
class PublicUserEducationDto {
  institution: string;
  institutionUrl: string | null;
  studyLevel: string | null;
  fieldOfStudy: string | null;
  startDate: Date | null;
  endDate: Date | null;
  isCurrent: boolean;
  description: string | null;
}
```

### Study Level Values
- `HIGH_SCHOOL`
- `ASSOCIATE`
- `BACHELOR`
- `MASTER`
- `DOCTORATE`
- `BOOTCAMP`
- `OTHER`
