# `GET /user/links`

Returns the social links you have entered on your profile.

## Basic Information
- type: `REST`
- method: `GET`
- url: https://api.remoet.dev/user/links
- returns: `JSON`

## cURL
```bash
curl --location 'https://api.remoet.dev/user/links' --header 'Authorization: Bearer <Your key>'
```

## HTTP
```http
GET /user/links HTTP/1.1
Host: api.remoet.dev
Authorization: Bearer <Your key>
```

## Javascript Fetch
```javascript
const myHeaders = new Headers();
myHeaders.append("Authorization", `Bearer ${key}`);

const requestOptions = {
  method: "GET",
  headers: myHeaders
};

fetch("https://api.remoet.dev/user/links", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.error(error));
```

## DTO

```typescript
class PublicUserLinksDto {
  url: string | null;
  githubUrl: string | null;
  facebookUrl: string | null;
  twitterUrl: string | null;
  linkedinUrl: string | null;
  youtubeUrl: string | null;
}
```

## Response

```json
{
    "url": "https://example.dev",
    "githubUrl": "https://github.com/you",
    "facebookUrl": null,
    "twitterUrl": "https://twitter.com/you",
    "linkedinUrl": null,
    "youtubeUrl": null
}
```

Fields you haven't filled in return `null`.

## Non authenticated response
```json
{
    "statusCode": 401,
    "message": "Invalid API Key",
    "error": "Unauthorized"
}
```
