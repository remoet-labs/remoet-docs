# `GET /user/linktrees`

Returns every link tree you've created.

## Basic information

- method: `GET`
- url: `https://api.remoet.dev/user/linktrees`
- returns: `JSON`

## cURL

```bash
curl -H "Authorization: Bearer <Your key>" https://api.remoet.dev/user/linktrees
```

## HTTP

```http
GET /user/linktrees HTTP/1.1
Host: api.remoet.dev
Authorization: Bearer <Your key>
```

## JavaScript fetch

```javascript
const res = await fetch("https://api.remoet.dev/user/linktrees", {
  headers: { Authorization: `Bearer ${process.env.REMOET_API_KEY}` },
});
const trees = await res.json();
```

## DTO

```typescript
class PublicLinkTreeDto {
  id: string;
  title: string;
  description: string | null;
  slug: string;
  links: Array<{
    id: string;
    label: string;
    url: string;
  }>;
  createdAt: Date;
  updatedAt: Date;
}
```

## Response

```json
[
  {
    "id": "65fa…",
    "title": "Side projects",
    "description": null,
    "slug": "side",
    "links": [
      { "id": "65fb…", "label": "GitHub", "url": "https://github.com/you" }
    ],
    "createdAt": "2026-04-01T12:00:00Z",
    "updatedAt": "2026-04-12T08:30:00Z"
  }
]
```

View and click counts are tracked server-side but are not exposed on the public API.

## `GET /user/linktrees/:slug`

Returns one link tree by its slug.

### Basic information

- method: `GET`
- url: `https://api.remoet.dev/user/linktrees/:slug`
- returns: `JSON`

### cURL

```bash
curl -H "Authorization: Bearer <Your key>" https://api.remoet.dev/user/linktrees/side
```

### Response

Same shape as a single element in the list above:

```json
{
  "id": "65fa…",
  "title": "Side projects",
  "description": null,
  "slug": "side",
  "links": [
    { "id": "65fb…", "label": "GitHub", "url": "https://github.com/you" }
  ],
  "createdAt": "2026-04-01T12:00:00Z",
  "updatedAt": "2026-04-12T08:30:00Z"
}
```

### Not found

```json
{
  "statusCode": 404,
  "message": "Link tree not found",
  "error": "Not Found"
}
```
