# About

remoet.dev is a job-tracking platform for developers. We surface open tech roles at companies you've explicitly chosen to follow, and we expose that data through both an MCP server and a REST API so your AI agents and your own tools can work with it directly.

The consumer site lives at [www.remoet.dev](https://www.remoet.dev). Source for these docs is at [github.com/remoet-labs/remoet-docs](https://github.com/remoet-labs/remoet-docs).

Found a stale claim or something missing? Open an issue on the docs repo or ping us via the consumer site's Discord link.
